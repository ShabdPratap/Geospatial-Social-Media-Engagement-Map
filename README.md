# Geospatial-Social-Media-Engagement-Map
This project visualises city-level social media engagement using Tableau. Workflow-Cleaned and aggregated social media engagement data by city-Geocoded cities using OpenStreetMap (Nominatim API)-Created an interactive Tableau dashboard colored by engagement level  ## Tools Used - Python (Pandas, Requests)- OpenStreetMap Nominatim API-Tableau Public
